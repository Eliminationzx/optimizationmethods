Метод покоординатного спуска c минимизацией по направлению


Общие сведения о методе

Поиск минимума функции в данном методе происходит путем нахождения минимального значения функции одной переменной.
Сначала, в функции F(X1; X2) фиксируется переменнaя X2, и происходит поиск минимума F(X1; X2) вдоль оси X1.
Затем фиксируется X1 и происходит поиск минимума F(X1; X2) вдоль оси X2.

Алгоритм

Аргументы алгоритма

F(X1; X2) - значение функции в соответствующих точках
E - точность вычислений

Последовательность шагов

  • Шаг1
    • Вычисляем минимум целевой функции F(X1; X2) вдоль оси X1 (идет оптимизация одной переменной).
  • Шаг2
    • Вычисляем минимум целевой функции F(X1; X2) вдоль оси X2 (идет оптимизация одной переменной).
  • Шаг3
    • Если max(dF, dX) < E, то завершаем процесс минимизации. Иначе - переходим к шагу 1.
