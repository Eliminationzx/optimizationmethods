Метод наискорейшего спуска

Общие сведения о методе

    Вычисление градиента на каждом шаге, позволяющее все время двигаться в направлении наиболее быстрого убывания целевой функции.
    Согласно этому методу, после вычисления в начальной точке градиента функции в направлении антиградиента движутся до тех пор, пока функция убывает.
    Достигнув точки минимума на выбранном направлении, снова вычисляют градиент функции и повторяют описанную процедуру.


Алгоритм

Аргументы алгоритма

F(X1; X2) - целевая функция;
λ - длина шага;
S - направление оптимизации (-∇F(Xk));
E - точность вычислений.

Последовательность шагов

  • Шаг1
    Выбираем направление для минимизации как антиградиент в исходной точке с координатами (X1; X2) для функции F(X1; X2).
  • Шаг2
    Вводим координаты градиента равные частным производным функции F(X). Если уже введены, то переходим к шагу 3.
  • Шаг3
    Если длина антиградиента меньше E, то минимум найден. Иначе - переходим к шагу 4.
  • Шаг4
    Вычисляем длину шага λ = min F(Х + λ*S) (Ищется любым методом одномерной оптимизации).
  • Шаг5
    Делаем шаг и принимаем новую точку. Переходим к шагу 1.

Особенности ввода производных

  • Переменные имеют форму записи "x1", "x2" (английская раскладка);
  • Для обозначения степени используется символ "^" ;
  • Символ "^" используется везде где он необходим;
  • Показатель степени следует записывать сразу после символа "^" (не допускаются пробелы);
  • Знак умножения используется везде где он необходим.
