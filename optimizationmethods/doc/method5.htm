Метод Нелдера - Мида

Общие сведения о методе

    Суть метода заключается в последовательном перемещении и деформировании симплекса вокруг точки экстремума.
    Вершина с наибольшим значением функции отражается через центр тяжести всех вершин на определенное расстояние. Отбрасывается вершина с наибольшим значением функции и процедура продолжается. Вершина с наибольшим значением функции отражается через центр тяжести оставшихся вершин на соответствующее расстояние. При этом используются операции Растяжения, Сжатия, Отражения и Редукции.

    Задаём начальные значения Хi, являющиеся вершинами треугольника (i = 1,2,3). В каждой из вершин многогранника вычисляем значения функции F(Xi), где i = 1,2,3.

Введём обозначения точек:

  • Xh – вершина, в которой функция принимает максимальное значение;
  • Xl – вершина, в которой функция принимает минимальное значение;
  • Xm – третья вершина;
  • Xc – центр тяжести всех вершин, кроме Xh.(Координаты Xc определяем по формуле: Xc = (Xm + Xl)/2)

Операция отражения

    Первой всегда выполняется операция отражения.
Точка Хотр – точка, полученная в результате операции отражения, вычисляется по формуле:
Хотр = Хc + a*(Хc – Хh), где a - коэффициент отражения (a = 1).
Вычисляем значение функции F(Хотр):

  • Если F(Xl) < F(Хотр) < F(Xm), то выполняем замену Xh на Xотр и выполняем операцию отражения.
  • Если F(Хотр) < F(Хl), то выполняем операцию растяжения.
  • Если F(Xm) < F(Хотр) < F(Xh), то выполняем операцию сжатия.
  • Если F(Хh) < F(Хотр), то выполняем операцию редукции.

Операция растяжения

    В результате получаем точку Храс, координаты которой вычисляются по формуле:
Храс = Хс + y*(Хотр – Хс), где y - коэффициент растяжения (y = 2).
Вычисляем значение функции F(Xрас):
Если F(Xрас) < F(Xl), то производим замену Хh = Xрас, иначе следует заменить Хh на Xотр.


Операция сжатия

    Точка Хсж, полученная в результате операции сжатия, вычисляется по формуле:
Хсж = Хс + b*(Xh – Xc), где b - коэффициент сжатия (b = 0.5).
Вычисляем значение функции F(Xсж): полагаем Хh = Xсж.



Операция редукции

    На этом шаге мы уменьшаем размерность симплекса, вершины, которого пересчитываются по следующей формуле:
Xi = Xl + 0.5*(Xi - Xl) для всех i (i = 1,2,3).
Редукция производится относительно точки в которой значение функции минимально.



Алгоритм

Аргументы алгоритма

F(X1,X2) - целевая функция;
E - точность вычислений;
Xh – вершина, в которой функция принимает максимальное значение;
Xl – вершина, в которой функция принимает минимальное значение;
Xc – центр тяжести всех вершин, кроме Xh.

Последовательность шагов

  • Шаг 1
    Выполняем операцию отражения относительно вершины, в которой функция принимает наибольшее значение по формуле Xотр = Xc + a*(Xc - Xh), где a - коэффициент растяжения. Переходим к шагу 2.
  • Шаг 2
    Если F(Xl) < F(Хотр) < F(Xm), то выполняем замену Xh на Xотр и переходим к шагу 1.
    Если F(Хотр) < F(Хl), то переходим к шагу 3.
    Если F(Xm) < F(Хотр) < F(Xh), то переходим к шагу 4.
    Если F(Хh) < F(Хотр), то переходим к шагу 5.
  • Шаг 3
    Выполняем операцию растяжения по формуле Xраст = Хс + y*(Xотр - Хс) , где y - коэффициент растяжения. Переходим к шагу 6.
  • Шаг 4
    Выполняем операцию сжатия по формуле Xсж = Xc + b*(Xh - Xc), где b - коэффициент сжатия. Переходим к шагу 6.
  • Шаг 5
    Выполняем операцию редукции.Вершины треугольника пересчитываем по следующей формуле: Xi = Xl + 0.5*(Xi - Xl) для всех i (i = 1,2,3). Переходим к шагу 6.
  • Шаг 6
    Если среднее расстояние от центра тяжести до вершин меньше требуемой точности, то завершаем процесс минимизации. Иначе - переходим к шагу 1.
