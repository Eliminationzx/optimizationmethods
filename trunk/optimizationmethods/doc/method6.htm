    В задачах поиска минимума произвольной квадратичной функции с положительной матрицей вторых производных метод Ньютона дает решение за одну итерацию независимо от выбора начальной точки. Однако в общем случае метод Ньютона может расходиться если начальное приближение находится вдали от точки минимума. Сходимость метода Ньютона можно гарантировать только в тех случаях, когда начальное приближение находится в достаточно малой окрестности точки минимума и матрица Гессе положительно определена и хорошо обусловлена. Поэтому на практике этот метод обычно используется в сочетании с одним из методов, быстро сходящимся вдали от точки минимума.

Особенности ввода производных

  • При вычислении производной функцию следует читать слева направо;
  • Переменные имеют форму записи "x1", "x2" (английская раскладка);
  • Для обозначения степени используется символ "^" ;
  • Знак умножения используется только между коэффициетом и скобкой;
  • Все скобки и коэффициенты внутри скобок необходимо записывать (например, если у вас получилась скобка, состоящая только из x1, то запись имеет вид "(x1-0)");
  • Вне скобок коэффициент 1 можно опускать и записывать только скобки и выражение внутри них (например, если у вас получилось выражение "1*(х1-1)", то запись имеет вид "(x1-1)");
  • Скобку, перед которой стоит коэффициент 0, можно не писать (например, если у вас получилось выражение "2*(х1-1)+0*(х2-0)", то запись имеет вид "2*(х1-1)");
  • Вне скобок коэффициент -1 можно опускать и записывать только знак минус, скобки и выражение внутри них (например, если у вас получилось выражение "-1*(х1-1)", то запись имеет вид "-(x1-1)");
  • При нахождении производной скобки раскрывать не нужно (в овражной функции находится как для сложной функции);
  • При вводе производной не допускаются пробелы.

Особенности ввода гессианов

  • Переменные имеют форму записи "x1", "x2" (английская раскладка);
  • Для обозначения степени используется символ "^" ;
  • Знак умножения пишется везде, где он необходим;
  • При вычислении гессиана овражной функции следует раскрыть скобки и расположить степени переменных по убыванию
  • При вводе обратного гессиана квадратичной функции следует сначала записывать его алгебраическое дополнение, затем через символ "/" определитель гессиана.
  • При вводе гессианов не допускаются пробелы.
